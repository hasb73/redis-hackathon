# Anomaly Injection Test Datasets

## Overview
Comprehensive test datasets with varying anomaly rates for system evaluation and benchmarking.

## Files
- `hdfs_eval_test_5.jsonl` - 5% anomaly rate evaluation dataset
- `hdfs_eval_test_10.jsonl` - 10% anomaly rate evaluation dataset
- `hdfs_eval_test_15.jsonl` - 15% anomaly rate evaluation dataset
- `hdfs_eval_test_40.jsonl` - 40% anomaly rate evaluation dataset

## Dataset Structure
Each JSONL file contains:
- **Log Entry**: Raw HDFS log text
- **Label**: Ground truth (0=normal, 1=anomaly)
- **Metadata**: Injection parameters and context

## Usage
These datasets are used by the evaluation module:
```bash
python3 evaluation/anomaly_evaluation.py evaluation/anomaly-injection-tests/hdfs_eval_test_40.jsonl
```

## Test Categories
- **Rate Testing**: Systematic evaluation at different anomaly percentages
- **Stress Testing**: Challenging scenarios with complex anomaly patterns
- **Balanced Testing**: Representative real-world anomaly distributions

## Generation
Datasets are automatically generated by `hdfs_anomaly_injection_loader.py` with configurable parameters for reproducible testing.
